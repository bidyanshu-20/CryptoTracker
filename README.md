CryptoTracker Web Application

A modern, cryptocurrency tracking web application built with React.js. CryptoTracker allows users to monitor real-time prices, track favorite coins, explore platform features, and understand different pricing plans, all from a single interface. The platform emphasizes ease of use, clean design, and accessibility, making it ideal for crypto enthusiasts, investors, and beginners alike.

Project Overview -->

CryptoTracker is designed to provide users with instant access to cryptocurrency data. It integrates multiple dashboards and functional components to give a complete user experience:

Real-Time Coin Tracking: Monitor live prices and coin trends.

Feature Insights: Understand what the platform offers to users.

Pricing Dashboard: View available subscription plans and choose the right one.

Blog Section: Stay updated with crypto-related articles.

User Authentication (Sign-Up): Create accounts and personalize tracking preferences.

The project is frontend-only at this stage, with user sign-ups stored locally in the browser via localStorage, making it a beginner-friendly full-feature simulation.

Detailed Features by Dashboard

1. Home Dashboard

The Home Dashboard is the landing page of CryptoTracker:

Top Cryptocurrencies List: Shows top coins with current prices, market cap, and 24-hour changes.

Currency Selector: Users can choose their preferred currency (USD, EUR, INR, GBP, JPY, AUD), and all prices update dynamically.

Navigation: Quick access to Coin Details, Features, Pricing, Blog, and Sign-Up pages.

Responsive Design: Optimized for desktop, tablet, and mobile devices.

2. Coin Details Dashboard

The Coin Dashboard provides detailed information about individual cryptocurrencies:

Coin Overview: Displays current price, market cap, volume, and rank.

Price Chart: (Future Enhancement) Visualizes historical prices over days, weeks, or months.

Quick Actions: Links back to the home page or other coins.

Clean Layout: Ensures readability and ease of comparison across coins.

3. Features Dashboard

The Features Dashboard highlights all the benefits and functionalities of CryptoTracker:

Human-Readable Descriptions: Each feature is explained clearly for users.

Interactive Layout: Cards or sections for each feature for easy scanning.

Examples of Features:

Real-time crypto prices

Currency conversion

Personalized tracking (via Sign-Up)

Alerts and notifications (planned for future)

4. Pricing Dashboard

The Pricing Dashboard displays subscription plans for advanced features:

Plan Comparison: Users can easily compare the features of Free, Standard, and Premium plans.

Readable Tiers: Each plan has a clear title, price, and feature list.

Call-to-Action: Encourages users to sign up for the selected plan.

Visual Design: Boxes/cards for each plan with hover effects to enhance user interaction.

5. Blog Dashboard

The Blog Dashboard keeps users updated with cryptocurrency-related content:

List of Articles: Titles, summaries, and links for detailed reading.

Navigation: Easy access back to other dashboards.

Future Enhancement: Embed videos, charts, or interactive tutorials.

6. Sign-Up / User Authentication

The Sign-Up feature allows users to create accounts and personalize their experience:

Form Fields: Username, Email, Password.

Validation: Prevents duplicate emails and ensures proper input.

LocalStorage Integration: Saves registered users in the browser for simulation purposes.

Navigation: Direct link to login page (planned).

Responsive Design: Works perfectly across devices.

7. Navbar & Footer

The Navbar and Footer provide seamless navigation and consistent branding:

Navbar: Links to Home, Features, Pricing, Blog, Sign-Up.

Currency Selector: Dynamically updates prices across the app.

Sign-Up Button: Easily accessible for new users.

Footer: Branding, links, and copyright information.

8. Modern UI & UX

CryptoTracker prioritizes visual appeal and usability:

Gradient Backgrounds and Fixed Background Images.

Semi-Transparent Cards with frosted glass effect for readability.

Smooth Hover Effects for buttons and cards.

Consistent Typography and spacing.

Technologies Used

Frontend: React.js, React Router

Styling: CSS, Flexbox, Grid, Gradient backgrounds

State Management: React Context API

Data Storage: LocalStorage for Sign-Up simulation

Assets: PNG & SVG images
